Viewers

A number of online previewers are available:
  • BT1 browser specifically for data taken by the BT1 instrument - not useful for any other instrument
  • Database search find files by date, comment, filename, etc.
  • Experiment browser for instruments running NICE and DCS and HFBS, a searchable and sortable index of experiments by user, experiment ID, title, etc.
  • ICP browser for data from SPINS, BT4, and old data from reflectometers
  • BT7 browser, which actually plots column-format data from BT7, MACS, PHADES, etc.
  • HDF5 viewer, which allows inspection of Nexus HDF5 data from any instrument that writes those (NICE instruments)
  • nexus-zip viewer, which allows inspection of Nexus-ZIP format data from any instrument that writes those (NICE instruments)

Helpers

API endpoints are provided for retrieving JSON data from the database about files:
  • ncnrdata/findRxCycle.php
    • optional parameter: timestamp, e.g. https://ncnr.nist.gov/ncnrdata/findRxCycle.php?timestamp=1321321231
    • returns rxcycle that matches current timestamp, or provided timestamp
  • https://ncnr.nist.gov/ncnrdata/listRxCycles.php
    • optional parameters: start_timestamp, end_timestamp
    • returns list of all rxcycles (name, start_date and end_date) that fall within start_timestamp and end_timestamp
    • https://ncnr.nist.gov/
  • https://ncnr.nist.gov/ncnrdata/listftpfiles.php
    • required paramter: pathlist[] e.g. ["cgd", "201809", "21237", "data"], defaults to [""]
    • accepts POST or GET
    • returns file and directory metadata for the provided pathlist, including
      • a list of subdirectories
      • (filename, last modified time, sha256) for all data files in pathlist
