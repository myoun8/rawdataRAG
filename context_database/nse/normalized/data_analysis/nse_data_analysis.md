# DAVE — Data Analysis and Visualization Environment

DAVE is an integrated environment for the reduction, visualization, and analysis of inelastic neutron scattering data. It is built using IDL and was intended as a cross-platform application for Linux, Windows, and Mac OS X. The page describes DAVE as available at no cost in two forms: a binary executable with an embedded IDL runtime license, and full source code for users with an IDL development license.

The main objective is to provide a user-friendly tool for neutron-scattering researchers to reduce, visualize, and interpret data. The page notes support for several neutron-scattering spectrometers at NCNR and PSI.
