# VSANS Instrument Publications

## Purpose

This page lists publications reporting measurements using the VSANS instrument and provides citation and acknowledgement guidance for VSANS-related papers and presentations.

## Citation and acknowledgement guidance

- For papers or presentations reporting measurements using the vSANS instrument, include the required acknowledgement to NSF and to NIST when there are no NIST coauthors. See [CHRNS publications and acknowledgement guidance](https://www.nist.gov/ncnr/chrns/publications-and-acknowledgment).
- Cite the VSANS instrument paper: [“The very small angle neutron scattering instrument at the National Institute of Standards and Technology”](http://dx.doi.org/10.1107/S1600576722000826), Barker J. et al., *J. Appl. Cryst.* **55**(2), 271 (2022).
- If data were reduced and/or analyzed using the Igor macros, cite [“Reduction and Analysis of SANS and USANS Data using Igor Pro”](http://dx.doi.org/10.1107/S0021889806035059), Kline, S. R., *J. Appl. Cryst.* **39**(6), 895 (2006).
- If data were analyzed using SasView, follow the [SasView publications guidance](https://www.sasview.org/publications/).
- Papers with NIST coauthors must be reviewed and approved by the Washington Editorial Review Board (WERB). WERB review includes attention to scientific merit, measurement uncertainties, SI units, and brand names. See [NIST technical publication policy / WERB](https://www.nist.gov/ncnr/nist-technical-publication-policy-werb).
